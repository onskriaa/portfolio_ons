Quiz sur la programmation

1. Quel langage de programmation est principalement utilisé pour le développement web côté client ?

a) Java b) Python c) JavaScript

2. Quelle balise est utilisée pour lier un fichier CSS à un fichier HTML ?

a) <css> b) <style> c) <link>

3. Qu'est-ce que HTML représente ?

a) HyperText Markup Language b) High-Level Text Language c) Hyper Transfer Markup Language

4. En programmation, que signifie l'acronyme "API" ?

a) Advanced Programming Interface b) Application Programming Interface c) Advanced Program Instruction

5. Quel est le symbole utilisé pour déclarer une variable en JavaScript ?

a) # b) @ c) $

6. Quelle méthode est utilisée pour ajouter un élément à la fin d'un tableau en JavaScript ?

a) append() b) push() c) addToEnd()

7. Quelle est la principale fonction de CSS ?

a) Gérer les bases de données b) Styler les éléments HTML c) Gérer les interactions utilisateur

8. Quel langage de programmation est souvent utilisé pour le développement d'applications mobiles ?

a) Swift b) C++ c) Kotlin

9. Quelle est la signification de l'acronyme "SQL" ?

a) Structured Query Language b) Simple Question Language c) System Query Language

10. Quelle est la différence entre `==` et `===` en JavaScript ?

a) Aucune différence b) `==` compare la valeur, `===` compare la valeur et le type de données c) `===` compare la valeur, `==` compare la valeur et le type de données
